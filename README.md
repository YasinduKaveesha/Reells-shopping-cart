# Grocery_SHOP Frontend

Initial React + Vite + TypeScript + Tailwind CSS frontend shell for the Grocery_SHOP shopping cart application.

## Routes

- `/` - Splash page
- `/home` - Landing/Home page
- `/shop` - Shop page with left persistent cart panel and right shopping workspace

## Run Locally

```bash
npm install
npm run dev
```

The frontend runs at:

```text
http://localhost:5173
```

## Current Scope

- Uses mock category, product, and cart data.
- No backend connection yet.
- No auth, checkout, payment, admin, AI recommendations, or analytics yet.
